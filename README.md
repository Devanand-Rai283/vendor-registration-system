# Vendor Registration System

**Full-Stack Web Application**

This project is a full-stack vendor registration system with:
- Frontend built using **HTML, CSS, JavaScript**
- Backend built using **Python (Flask)**


## 🛠 Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Python (Flask)

**Database**
- SQLite / MySQL (if used)


## ⚙️ Architecture

- Frontend handles user input and UI
- Backend (`app.py`) processes requests and validations
- Data is stored and managed via backend logic
